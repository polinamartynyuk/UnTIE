# Эксперименты

Исследовательские ноутбуки пока сохранены в `scripts/`, чтобы не ломать их относительные пути. Новые ноутбуки размещайте в `experiments/notebooks/` и импортируйте пакет `untie`, а не копируйте классы и функции в ячейки.

Канонические исполняемые сценарии:

- `python -m untie.cli ... --mode baseline`
- `python -m untie.cli ... --mode attention --reference-answer ...`

Legacy-ноутбуки и старые нумерованные Python-скрипты не входят в поддерживаемый API пакета.
